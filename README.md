Code Guide

Overview

Code Guide is a web-based platform to help programmers and students learn coding through structured tutorials and practical examples. Currently, it is built entirely with frontend technologies, with plans to integrate a Java backend in the future for dynamic content and enhanced functionality.

Features

Language Tutorials: Learn programming basics with clear, step-by-step examples.

Code Snippets: Copy and run code examples instantly in your browser.

Mini Projects: Practice projects to strengthen coding skills.

Responsive Design: Works on both desktop and mobile devices.

Future Backend Integration: Java Spring Boot backend planned for handling dynamic content, user management, and project submissions.


Tech Stack

Frontend: HTML, CSS, JavaScript

Future Backend: Java (Spring Boot)

Deployment: Static hosting for frontend now (Netlify, GitHub Pages), backend integration later
